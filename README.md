# simple_ocr
一个简易的识别验证码的代码
我的博客：https://blog.csdn.net/qq_44749634?type=blog
这是一个简易验证码识别系统，目的是搭建一个验证识别服务
现在还是很简陋的，且识别率较低，且普适性较差
