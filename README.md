# simple_ocr
一个简单的识别验证码的代码
我的博客：https://blog.csdn.net/qq_44749634?type=blog
我的联系方式：QQ 738389368
这是一个基于dajngo api 做的验证码识别
现在还是很简陋的，且识别率较低，且普适性
如果某些验证码识别率很低，请联系我改进！
